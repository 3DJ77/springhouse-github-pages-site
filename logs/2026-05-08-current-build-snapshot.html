Current Build Snapshot — All Four Modules Standing

build snapshot greenhouse coop solar field testing site work

Quick site update. The greenhouse module is standing and in field testing on the property. Alongside the container module, the chicken coop module, and the chicken run bridge module — all four main site structures are now up.

The Four Modules

Springhouse is organized around four structural modules on the site. Getting all four standing has been the primary physical milestone for this phase of the build:

  • Container module. The steel container provides secure covered storage and a weather-protected work surface. Swale excavation around it for runoff control is underway.
  • Chicken coop module. Moved to its new location. The curved canopy conversion using greenhouse plastic is in progress — old flat roof is off, new curved frame is test-fit and ready for the plastic cover.
  • Chicken run bridge module. Connects the coop to the run area. In place.
  • Greenhouse module. Standing and in active field testing. The greenhouse is the most recently completed module and the one with the most active work happening around it right now.

Greenhouse Field Testing

Having a greenhouse module standing is different from having it working. Field testing means putting it through actual conditions — checking how it handles temperature swing, where condensation builds up, how airflow behaves during the day, and what the structure does under wind load. None of that shows up in a plan or a dry-run assembly. It only shows up once the structure is in place and the weather is doing what weather does.

More detail on specific findings from greenhouse field testing will follow in the next posts as the data comes in.

Power Status

The solar harvest and battery bank are running steady:

  • Active harvest: 700W total — 500W on the 24V system (5-panel string on temporary ground mount), 200W on the primary 12V system.
  • Battery bank: Two 8D Duracell lead-acid batteries on the 24V bank. Topped up and holding well — recovery from winter is complete.
  • Tool charging: Operational. Cordless tools charging directly from solar. This has been running without issues since the first test on April 26.
  • Full capacity plan: 2,800W panel capacity once the ground mount is in its current setup. The temporary mount has been working fine in the interim.

Site State Summary

  • Container: in place, swale work ongoing
  • Run bridge: in place
  • Fence posts: in ground
